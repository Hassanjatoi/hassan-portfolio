
# Hassan Jatoi - Portfolio (static)

Files inside this ZIP are ready to be uploaded to GitHub and published via GitHub Pages.
To publish:
1. Create a new GitHub repository (e.g., `hassan-portfolio`).
2. Upload all files and folders from this ZIP to the repository root (index.html must be in root).
3. Commit and push.
4. In repository Settings → Pages → Source, choose the branch (main/master) and folder (root).
5. Save — your site will be published at `https://<your-username>.github.io/<repo-name>/` within a minute or two.

You can also deploy on Netlify or Vercel by dragging the folder to those dashboards.
